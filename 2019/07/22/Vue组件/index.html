Vue组件

组件注册

1、全局注册

优点：所有的Vue实例都可以使用；缺点：权限太大，容错率较低，容易被覆盖

<div id="root">
    <my-component></my-component>
</div>
<script>
    Vue.component('my-component', {
        template: '<div>这是一个全局组件</div>',
    }) //全局注册
    var vm = new Vue({
        el: '#root',
        data: {}
    })
</script>

2、局部注册

组件的特点：

1、JS中的驼峰命名会转换为HTML中的小写字母+-(HTML不支持大写)的形式，反之亦然

2、template中的内容必须被一个DOM元素包裹(顶级元素只能有一个)

3、组件中也可以定义data(必须是一个方法)、computed、methods等属性

<div id="root">
    <app-component></app-component>
    <app-component></app-component>
</div>
<script>
    var appComponent = {
        template: '<button @click="handle">{{count}}</button>',
        data() {
            return { count: 0 }
        }, //子组件的data属性必须是一个方法
        methods: {
            handle() { this.count++ }
        }
    }
    var vm = new Vue({
        el: '#root',
        components: {
            appComponent //局部注册，使用ES6简写形式
        }
    })
</script>

html限制：table中放div组件会被提到外面，可以通过is属性来突破限制实现挂载

<table>
    <tbody is="app-component"></tbody>
</table>

父组件向子组件传值

传值规则

1、子组件通过props属性(字符串数组)接收父组件传递的参数

2、props中的数据来自父级，而组件data中return的是组件自己的数据

两者的作用域均为组件本身，可以在template、computed和method中直接调用

3、props的值有两种形式，一种字符串数组(由属性名组成)；一种是对象，用来进行数据类型限定

单向数据流

父组件数据变化可以传递给子组件，但是反过来就不行

目的是尽可能将父子组件解耦，避免子组件无意中修改了父组件的状态

<div id="app">
    <child-component :child-msg="msg"></child-component>
</div>
<script>
    var childComponent = {
        props: ['childMsg'],
        template: '<div>{{childMsg}}</div>'
    }
    var vm = new Vue({
        el: '#app',
        components: { childComponent },
        data: { msg: 'hello star!我在父组件中' }
    })
</script>

属性类型校验

props:{ 
    childMsg: Number, //指定单一类型
    age: [String, Number], //指定类型范围
    name: { 
        type: String, //类型
        default: 'star', //默认值
        required: true, //必传
        validator(value){ return value.length < 10 } //自定义验证规则
    } 
}

子组件向父组件传值

自定义事件，在子组件中用$emit触发事件，在父组件中监听事件，实现子组件给父组件传递数据

<div id="app">
    <h1>{{total}}</h1>
    <child-component @change="handleTotal"></child-component>
</div>
<script>
    var childComponent = {
        template: `
            <div>
                <button @click="handleIncrease">加一</button>
                <button @click="handleReduce">减一</button>
            </div>`,
        methods: {
            handleIncrease() { this.$emit('change', 1) },
            handleReduce() { this.$emit('change', -1) }
        }
    }
    var vm = new Vue({
        el: '#app',
        components: { childComponent },
        data: { total: 2000 },
        methods: {
            handleTotal(value) { this.total += value }
        }
    })
</script>

使用v-model实现上述功能，是一个语法糖(v-model绑定了input事件)

<div id="app">
    <h1>{{total}}</h1>
    <child-component :total="total" v-model="total"></child-component>
</div>
<script>
    var childComponent = {
        props: ['total'],
        template: `
            <div>
                <button @click="handleIncrease">加1000</button>
            </div>`,
        data() {
            return { count: this.total }
        },
        methods: {
            handleIncrease() { this.$emit('input', this.count += 1000) }
        }
    }
    var vm = new Vue({
        el: '#app',
        components: { childComponent },
        data: { total: 2000 }
    })
</script>

插槽slot

在子组件中定义插槽位置，在父组件中向插槽中插入内容

<div id="app">
    <child-component>
        <p slot="paragraph">hello world</p>
    </child-component>
</div>
<script>
    var childComponent = {
        template: `
            <div>
                <slot name="paragraph">这是子组件插槽默认内容</slot>
            </div>`
    }
    var vm = new Vue({
        el: '#app',
        components: { childComponent }
    })
</script>
