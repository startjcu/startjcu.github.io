区别几个写入内容API

innerText / innerHTML

前者向元素中写入纯文本

后者向页面中写入html文本(会将html标签进行转义)

textContent / innerText

前者获取的是源html中的文本

后者能获取html文档渲染后文档中的文本(如元素中若使用了text-transform样式后的文本)

循环中绑定事件

循环中的变量是容器

而事件绑定是异步的，会等循环执行完后才执行

所以直接在事件函数中使用变量，使用的是循环结束时的变量

这种情况下应该使用e.target来获取目标元素，才能绑定成功
