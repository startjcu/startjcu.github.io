响应式和Flex布局

发表于 2019-03-09

媒体查询

若只用max-width，要注意书写顺序，像素大的往后放

国内很少用响应式，一般用两个域名，或是用两套html和css

苹果手机默认用375像素的宽度去模拟980像素，然后进行缩放，故需要设置viewport

手机端交互方式的特点

1、没有hover，2、有touch事件，3、没有resize，4、没有滚动条

Flex布局

Flex之前的布局

1、文档流（正常流 Normal Flow）； 2、float + clear；

3、position：relative + absolute； 4、display：inline-block； 5、负margin

Flex布局特点

1、块级元素侧重垂直方向、行内布局侧重水平方向，flex布局与方向无关

2、flex布局可以实现空间自动分配、元素自动对齐（flexible：弹性、灵活）

3、flex适用于简单的线性布局，更复杂的布局要用到grid布局

Flex基本概念

主轴(main axis)：主尺寸(main size)、主轴起点(main start)、主轴终点(main end)

侧轴(cross axis)：侧尺寸(cross size)、侧轴起点(cross start)、侧轴终点(cross end)

子元素(flex item)、父元素(flex container需添加display: flex样式)

flex container的属性

1、方向：flex-direction

属性值：row默认值、column、row-reverse、column-reverse

2、换行：flex-wrap

属性值：nowrap默认值、wrap、wrap-reverse

3、流动：flex-flow

是方向和换行的复合属性：flex-flow = flex-direction + flex-wrap

4、主轴方向对齐方式：justify-content

属性值：flex-start默认值、space-between、center、space-around、flex-end

5、侧轴方向对齐方式：align-items

属性值：stretch默认值(将所有元素拉伸至最高元素的高度)、flex-start、center、flex-end

6、行与行之间隔多远：align-content

(很少用)属性值：stretch默认值、flex-start、flex-end、space-around、space-between

flex item的属性

1、增长比例：flex-grow

空间过多时，多余空间分配比例

2、收缩比例：flex-shrink

空间不够时，所缺空间分摊比例，一般不用

3、默认大小：flex-basis

一般不用，原始大小由内容大小决定，无需特意指定

4、复合属性：flex

flex = flex-grow + flex-shrink + flex-basis

5、属性：order

改变子元素排列顺序，默认值为0

6、自身的侧轴对齐方式：align-self

用于改变单独某个子元素的侧轴对齐方式，属性值同父元素的align-items属性
