命令行基础

发表于 2019-01-04

路径符号

用户目录：~

整个硬盘：/
故以/开头的路径为绝对路径

当前目录：.；父级目录：..

常用单词

罗列：list

改变：change

目录：directory

文件：file

创建：make

删除：remove

移动：move

复制：copy

链接：link

查找：find

触摸：touch

回音：echo

命令是由常用单词组合的缩写，缩写规则是删掉元音字母，保留前2~3个辅音字母

常用命令

创建目录：mkdir

全称：make directory

-p：parents，创建多级目录

删除：rm

全称：remove

-r：recursive，递归删除，删除文件夹必加参数

-f：force，强制删除，取消逐级递归提示是否删除

复制：cp

全称：copy

-r：recursive，递归复制，复制文件夹必加参数

移动或重命名：mv

全称：move

列出文件：ls

全称：list

-l：long，以较长的格式列出文件信息

-a：all，列出当前目录下所有文件的信息

改变目录：cd

全称：change directory

打印当前路径：pwd

全称：print working directory

磁盘文件系统：df

全称 disk file system

-h：human-readable

当前目录使用情况：du

全称：disk usage

命令行url：curl

全称：command uniform resource locator

创建/更新文件：touch

编辑文件：echo

> 写入内容，会覆盖之前的内容；>> 追加内容；>! 实现在mac上向文件写入内容

命令行中的空格具有特殊含义(用于分隔命令、参数和值)，所以带空格的字符串一定要用''引起来

命令资源

1、查看手册：man cName / cName --help

2、超级好用的命令详解网站

3、查看常用的使用文档:tldr cName (too long didn't read)

Git的使用

将远程存仓库的代码克隆到本地

以简短的形式显示有变更的文件(short branch)

将当前目录下的变动提交到缓存区

将代码提交到本地仓库并添加注释

拉取远程代码，同步远程的变动

将本地代码提交到远程仓库

查看提交日志

脚本script

脚本即一段程序，让计算机按照一定的顺序执行一系列命令

JavaScript是一门动态类型，面向对象的脚本语言

通过在文件首行添加#!/usr/bin/env node来指定在Linux下该文件默认以node命令执行

Markdown语法

发表于 2019-01-02

标题

标题：#代表一级标题，##代表二级标题，依此类推

分隔线：在一个空行中插入三个及以上的*号，除此外行内不能有其他内容

字体

倾斜：*倾斜*

加粗：**加粗**

中划线：~~中划线~~

倾斜加粗：***倾斜加粗***

代码

单行代码：反引号

console.log('hello world')

多行代码：三反引号

列表

  • 无序列表：* 无序列表
  1. 有序列表：1. 有序列表

表格

姓名 年龄 身高
star 18 168

表头：姓名|年龄|身高；标识：-|-|-；内容：star|18|168

链接

链接：<链接地址>或[链接名称](链接地址)
