HTTP基础知识

页面相关术语

URL：Uniform Resource Locator

统一资源定位符，用来定位页面

HTTP：HyperText Transfer Protocol

超文本传输协议，用来下载页面

HTML：HyperText Markup Language

超文本标记语言，用来显示页面

资源相关术语

URI：Uniform Resource Identifier

统一资源标识符，有两种实现方式URL和URN

URL：Uniform Resource Locator

统一资源定位符，用来定位一个资源，一般指代网址，但文件移动后就找不到了

URN：Uniform Resource Name

统一资源名称，用来唯一标识某个资源，如ISBN国际标准书号International Standard Book Number

URL的组成

协议：(Protocal)https://

域名：(Host)startjcu.com，其中com为顶级域名、startjcu为二级域名

端口：(Port):443，对应服务器上的某个Web服务，http-80为默认端口

路径：(Path)/resum

参数：(Parameter)?id=1&name=star

锚点：(Hash)#3，用来实现某个页面内不同位置的跳转

DNS服务

域名系统：Domain Name System，用来将域名解析为对应的IP地址

流程：域名 > (本地hosts)电信DNS服务 > IP地址

服务器(Server电脑) > HTTP > 客户端(Client浏览器)

HTTP的作用是置动服务器和浏览器之间如何通信

HTTP请求

curl -X POST -s -v -- 'https://www.baidu.com'

curl：command line url，此命令用于在命令行实现url请求

-X：request method，指定请求方法

-s：silence，不显示请求进度条

-v：verbose，详细地显示请求>和响应<

HTTP方法

GET：获取数据

POST：上传(提交)数据

PUT/PATCH：更新(整体/局部)数据

DELETE：删除数据

其他方法：HEAD、OPTIONS等

请求报文格式

1、请求方法 路径及参数 协议/版本号

2、key : value键值对，如Host: www.baidu.com

3、空行，用来分隔2和4

4、要上传的数据，如用户名、密码等

响应报文格式

1、协议/版本号 状态码 状态说明

2、key : value键值对，如Content-Type: 141

3、空行，用来分隔2和4

4、下载的数据

状态码
1xx：消息，如100-Continue，表示还需要进一步的操作

2xx：成功，如200-OK，表示普通成功；201-Created，表示创建成功

3xx：重定向，如301-Moved Permanently；302-Move Temporarily

4xx：请求错误，如401-Unauthorized未登录；404-Not Found资源丢失

连接相关术语

TCP：Transmission Control Protocol

传输控制协议，每次建立连接前，客户端和服务器都要进行三次成功对话后，才正式开始传输内容(TCP三次握手)

UDP：User Datagram Protocol

用户数据报协议，和TCP的区别是，TCP可靠，面向连接，相对UDP较慢，而UDP正好相反

IP：Internet Protocol

网络协议，每个电脑都应该有一个独立的IP地址。127.0.0.1/localhost指本机，0.0.0.0不指向任何设备

外网IP > 路由器 > 内网IP

路由器(必由之路)，用于给信息指路/进行内容分发，故又称网关

HTTP是基于TCP/IP协议构建的，使用HTTP协议访问另一个主机时，必须提供IP地址和端口号

小知识：argc=arguments count、argv = arguments value

命令行基础

路径符号

用户目录：~

整个硬盘：/
故以/开头的路径为绝对路径

当前目录：.；父级目录：..

常用单词

罗列：list

改变：change

目录：directory

文件：file

创建：make

删除：remove

移动：move

复制：copy

链接：link

查找：find

触摸：touch

回音：echo

命令是由常用单词组合的缩写，缩写规则是删掉元音字母，保留前2~3个辅音字母

常用命令

创建目录：mkdir

全称：make directory

-p：parents，创建多级目录

删除：rm

全称：remove

-r：recursive，递归删除，删除文件夹必加参数

-f：force，强制删除，取消逐级递归提示是否删除

复制：cp

全称：copy

-r：recursive，递归复制，复制文件夹必加参数

移动或重命名：mv

全称：move

列出文件：ls

全称：list

-l：long，以较长的格式列出文件信息

-a：all，列出当前目录下所有文件的信息

改变目录：cd

全称：change directory

打印当前路径：pwd

全称：print working directory

磁盘文件系统：df

全称 disk file system

-h：human-readable

当前目录使用情况：du

全称：disk usage

命令行url：curl

全称：command uniform resource locator

创建/更新文件：touch

touch fileName

编辑文件：echo

echo ‘content’ > fileName

> 写入内容，会覆盖之前的内容；>> 追加内容；>! 实现在mac上向文件写入内容

命令行中的空格具有特殊含义(用于分隔命令、参数和值)，所以带空格的字符串一定要用''引起来

命令资源

1、查看手册：man cName / cName --help

2、超级好用的命令详解网站

3、查看常用的使用文档:tldr cName (too long didn't read)

Git的使用

git clone

将远程存仓库的代码克隆到本地

git status -sb(short branch)

以简短的形式显示有变更的文件

git add .

将当前目录下的变动提交到缓存区

git commit -m 'comment'

将代码提交到本地仓库并添加注释

git pull

拉取远程代码，同步远程的变动

git push

将本地代码提交到远程仓库

git log

查看提交日志

script脚本

脚本即一段程序，让计算机按照一定的顺序执行一系列命令

JavaScript是一门动态类型，面向对象的脚本语言

通过在文件首行添加#!/usr/bin/env node来指定在Linux下该文件默认以node命令执行

Markdown语法

标题

标题：#代表一级标题，##代表二级标题，依此类推

分隔线：在一个空行中插入三个及以上的*号，除此外行内不能有其他内容

字体

倾斜：*倾斜*

加粗：**加粗**

中划线：~~中划线~~

倾斜加粗：***倾斜加粗***

代码

单行代码：反引号

console.log('hello world')

多行代码：三反引号

列表

  • 无序列表：* 无序列表
  1. 有序列表：1. 有序列表

表格

姓名 年龄 身高
star 18 168

表头：姓名|年龄|身高；标识：-|-|-；内容：star|18|168

链接

链接：<链接地址>或[链接名称](链接地址)
