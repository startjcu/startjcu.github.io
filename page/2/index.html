JS-AJAX

发表于 2019-02-08

AJAX

定义

异步的JavaScript和XML：Asynchronous Javascript And XML

使用

1、使用XMLHttpRequest发请求

2、服务器返回XML格式的字符串

3、JS解析XML，并更新局部页面

现在使用更轻量级的JSON代替XML

readyState：0 - open() - 1 - send() - 2 (获取响应头) - 3 (正在下载响应体) - 4 (请求完成)

同源策略

同源：协议、域名和端口均相同为同源

目的：保证用户信息的安全，防止恶意的网站窃取数据

CORS跨域

跨域资源共享：Cross-Origin Resource Sharing

AJAX请求是严格遵守同源策略的，若要突破同源策略，就需要用到CORS跨域

跨域请求

form表单请求源码

<img>发送请求

img发送请求源码

script发送请求源码

缺点：这种请求方式要求前端程序员对后端接口很清楚才能正确定义请求路径

JSONP

JSONP底层实现源码

在script的请求(src)后面添加callback查询参数

JSONP实现原理

1、请求方(一个网站的前端-浏览器)创建script，src指向响应方，同时传一个查询参数?callbackName=xxx

2、响应方根据查询参数callbackName，构造形如xxx.call(undefined,'你要的数据')，即xxx(result)的数据返回给客户端

3、浏览器接收到响应，就会执行xxx.call(undefined, ‘你想要的数据’)

4、浏览器将执行结果渲染出来，展示给用户，请求方就获得了他想要的数据

业界约定：1、callbackName一般用callback代替，2、xxx一般为随机数

jQuery实现JSONP

jQuey底层会封装请求参数无需单独指定

其他相关

1、JSONP为什么不支持POST请求

因为JSONP是通过动态创建script实现跨域请求的，只能用GET，不能用POST

2、JSONP与JSON无关

一般服务器端执行callback函数会将参数以JSON的格式传入，加上两端Padding，故称JSONP

3、JSONP与AJAX无关

因为jQuery将JSONP的实现封装在ajax属性/函数中，但AJAX的出现是在JSONP之后的

事件队列，可以给一个元素添加两个click事件，按定义顺序执行

JS-DOM

发表于 2019-01-30

DOM

文档对象模型：Document Object Model是HTML和XML文档的编程接口

Node接口

Node(节点)是一个接口，许多DOM API都会继承这个接口，如Document、Element、Text

Node常用属性

childNodes:获取所有子节点

firstChild：获取第一个子节点

lastChild：获取最后一个子节点

nodeName：通过该属性获取的是节点名是大写格式的，后加入的svg除外

nodeType：节点类型，1-Node.ELEMENT_NODE、3-Node.TEXT_NODE

parentNode：获取父节点

previousSibling：获取上一个兄弟节点

nextSibling：获取下一个兄弟节点

Node(节点)的类型很多，如注释节点、文本节点等，易忽略的是换行和空格也会构成文本节点

Node常用方法

appendChild()：添加子节点

cloneNode()：克隆节点，可以传入布尔值；默认浅拷贝，传入true后是深拷贝

isEqualNode()：判断两个节点是否相同

isSameNode()：判断两个节点是否是同一个节点

normalize()：标准化，常见的将同时添加的两个文本节点转化为一个文本节点

Document接口

指整个加载好的网页，是操作DOM树的入口

document是浏览器全局对象window的一个属性，原型是HTMLDocument.prototype

Document常用属性

body：获取body节点

characterSet：获取当前页面字符编码

doctype：获取文档类型

documentElement：获取文档元素，不包括doctype

location：获取网页URI

浏览器加载document的过程：open -> write -> close(页面加载完毕后自动执行)

所以如果异步向页面写入内容，会覆盖之前页面中的内容，因为有网页关闭后在重新开启的过程

Document常用方法

createElement()：创建一个元素节点，在HTML中元素特指标签

createTextNode()：创建一个文本节点

getElementById()：获取某个DOM节点

getElementsByClassName()、getElementsByTagName()：获取一组DOM节点

Element接口

特有属性

tagName：获取标签名，是大写格式的

classList：获取class属性伪数组，相关方法add()、remove()、toggle()

children：获取所有子元素

previousElementSibling：前一个兄弟元素

nextElementSibling：下一个兄弟元素

优化查找DOM元素的方法

querySelector()：获取匹配指定选择器的第一个元素

querySelectorAll()：获取匹配指定选择器的一组元素
