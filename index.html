命令行基础

路径符号

用户目录：~

整个硬盘：/
故以/开头的路径为绝对路径

当前目录：.；父级目录：..

常用单词

罗列：list

改变：change

目录：directory

文件：file

创建：make

删除：remove

移动：move

复制：copy

链接：link

查找：find

触摸：touch

回音：echo

命令是由常用单词组合的缩写，缩写规则是删掉元音字母，保留前2~3个辅音字母

常用命令

创建目录：mkdir

全称：make directory

-p：parents，创建多级目录

删除：rm

全称：remove

-r：recursive，递归删除，删除文件夹必加参数

-f：force，强制删除，取消逐级递归提示是否删除

复制：cp

全称：copy

-r：recursive，递归复制，复制文件夹必加参数

移动或重命名：mv

全称：move

列出文件：ls

全称：list

-l：long，以较长的格式列出文件信息

-a：all，列出当前目录下所有文件的信息

改变目录：cd

全称：change directory

打印当前路径：pwd

全称：print working directory

磁盘文件系统：df

全称 disk file system

-h：human-readable

当前目录使用情况：du

全称：disk usage

命令行url：curl

全称：command uniform resource locator

创建/更新文件：touch

touch fileName

编辑文件：echo

echo ‘content’ > fileName

>写入内容，会覆盖之前的内容；>>追加内容；>!mac上写入内容

命令行中的空格具有特殊含义，所以带空格的字符串一定要用''引起来

命令资源

1、查看手册：man cName/cName --help

2、超级好用的命令详解网站

3、查看精炼的使用文档:tldr cName(too long didn't read)

Git的使用

git clone

将远程存仓库的代码克隆到本地

git status -sb(short branch)

以简短的形式显示有变更的文件

git add .

将当前目录下的变动提交到暂存区

git commit -m 'comment'

将代码提交到本地仓库并添加注释

git pull

拉取远程代码，同步远程的变动

git push

将本地代码提交到远程仓库

git log

查看提交日志

script脚本

脚本即一段程序，让计算机按照一定的顺序执行一系列命令

JavaScript是一门动态类型，面向对象的脚本语言

通过在文件首行添加#!/usr/bin/env node来指定在Linux下该文件默认以node命令执行

HTTP基础知识

URL-HTTP-HTML间的关系

URL：Uniform Resource Locator

统一资源定位符，用来定位页面

HTTP：HyperText Transfer Protocol

超文本传输协议，用来下载页面

HTML：HyperText Markup Language

超文本标记语言，用来显示页面

URI-URL-URN间的关系

URI：Uniform Resource Identifier

统一资源标识符，有两种实现方式URL和URN

URL：Uniform Resource Locator

统一资源定位符，用来定位一个资源，一般指代网址，但文件移动后就找不到了

URN：Uniform Resource Name

统一资源名称，用来唯一标识某个资源，如ISBN国际标准书号International Standard Book Number

URL的组成

协议：(Protocal)https://

域名：(Host)startjcu.com，其中com为顶级域名、startjcu为二级域名

端口：(Port):443，对应服务器上的某个Web服务，http-80为默认端口

路径：(Path)/resum

参数：(Parameter)?id=1&name=star

锚点：(Hash)#3，用来实现某个页面内不同位置的跳转

DNS服务

域名系统：Domain Name System，用来将域名解析为对应的IP地址

流程：域名 > (本地hosts)电信DNS服务 > IP地址

服务器(Server电脑) > HTTP > 客户端(Client浏览器)

HTTP的作用是置动服务器和浏览器之间如何通信

HTTP请求

curl -X POST -s -v -- 'https://www.baidu.com'

curl：command line url，此命令用于在命令行实现url请求

-X：request method，指定请求方法

-s：silence，不显示请求进度条

-v：verbose，详细地显示请求>和响应<

HTTP方法

GET：获取数据

POST：上传(提交)数据

PUT/PATCH：更新(整体/局部)数据

DELETE：删除数据

其他方法：HEAD、OPTIONS等

请求报文格式

1、请求方法 路径及参数 协议/版本号

2、key : value键值对，如Host: www.baidu.com

3、空行，用来分隔2和4

4、要上传的数据，如用户名、密码等

响应报文格式

1、协议/版本号 状态码 状态说明

2、key : value键值对，如Content-Type: 141

3、空行，用来分隔2和4

4、下载的数据

状态码
1xx：消息，如100-Continue，表示还需要进一步的操作

2xx：成功，如200-OK，表示普通成功；201-Created，表示创建成功

3xx：重定向，如301-Moved Permanently；302-Move Temporarily

4xx：请求错误，如401-Unauthorized未登录；404-Not Found资源丢失

TCP-UDP-IP间的关系

TCP：Transmission Control Protocol

传输控制协议，每次建立连接前，客户端和服务器都要进行三次成功对话后，才正式开始传输内容(TCP三次握手)

UDP：User Datagram Protocol

用户数据报协议，和TCP的区别是，TCP可靠，面向连接，相对UDP较慢，而UDP正好相反

IP：Internet Protocol

网络协议，每个电脑都应该有一个独立的IP地址。127.0.0.1/localhost指本机，0.0.0.0不指向任何设备

外网IP > 路由器 > 内网IP

路由器(必由之路)，用于给信息指路/进行内容分发，故又称网关

HTTP是基于TCP/IP协议构建的，使用HTTP协议访问另一个主机时，必须提供IP地址和端口号

小知识：argc=arguments count、argv = arguments value

CSS归纳整理

概念

层叠样式表：Cascading Style Sheets

引入方式

1、行内样式：style="color:red;"

2、内部样式表：<style>*{ margin:0; padding:0; }</style>

3、外部样式表：<link rel="stylesheet" href="./css/a.css">

4、样式表中引入另一个样式表：@import url(./b.css)

浮动

1、给所有子元素添加float属性

2、给父元素添加clearfix类：.clearfix::after{ content:''; display:block; clear:both; }

文档流

Normal Flow，指HTML文档内元素的流动方向

1、行内元素，从左向右流动，遇到阻碍就另起一行

word-break: break-all将单词按字母打断，用于英文长单词遇到阻碍换行显示

2、块级元素，每个块独占一行，从上往下流动

display: inline-block尽量少使用，优先考虑用float: left代替

块级元素的高度问题

1、用height属性指定的高度

2、等于其内部文档流元素的高度总和

有些行内元素的高度可能大于它所在的块级元素的高度，因为字体有设计建议行高

行内元素内容居中问题

尽量少使用height和line-height

可考虑display:inline-block与padding来进行水平和垂直居中

绝对定位

1、给子元素添加绝对定位：display: absolute

2、给父元素添加相对定位：display: relative

绝对定位会使元素脱离文档流，定位基准为最近一个的已定位的父级元素的左上角

固定定位

给元素添加固定定位：display: fixed

固定定位会使元素脱离文档流，定位基准为浏览器左上角，定位后元素位置不会随页面的滚动变化

CSS布局

1、左右布局

使用浮动，左边元素左浮动，右侧元素右浮动，父级元素清除浮动

2、左右中布局

操作方式同1，左侧和中间元素左浮动，右侧元素右浮动，父级元素清除浮动

3、水平居中

行内元素，在父级元素上加text-align实现居中

块级元素，通过margin: 0 auto实现居中

4、垂直居中

通过指定height和line-height相对来居中

或者将元素的上下填充padding设置为等值进行居中

5、其他小技巧

伪元素：非空元素(可以有子元素的元素)默认都具有::before和::after伪元素
可以将其理解为一个隐藏的span，和生命周期钩子一样，但无法被选中进行复制操作

伪类：选择器的:nth-child(even)可以对偶数个子元素单独设置样式，实现更复杂的布局

对应浮动排版错乱的情况，使用border大法进行查看，看是否是因为由行高不一致导致浮动异常

CSS资源

CSS经典范例：CSS Tricks

iconfont图标：矢量图标库

其他资源：阮一峰CSS、张鑫旭CSS、Codrops、《CSS揭秘》

HTML常用标签

标题标签

headline：<h>

用来表示不同级别的标题<h1>~<h6>

段落标签

paragraph：<p>

用来表示一个段落

换行标签

break：<br>

用来实现网页上的换行

水平线标签

horizontal line：<hr>

用来在网页上画一条水平线

斜体

italic：<i>、emphasize：<em>

前者表示样式上倾斜，后者表示语气上强调

粗体

blod：<b>、strong：<strong>

同样的，前者是样式上加粗，后者是语气上加重

列表

无序列表：unordered list <ul>、列表项：list item <li>

<ul><li>item</li></ul>

有序列表：ordered list <ol>、列表项：list item <li>

<ol><li>item</li></ol>

定义列表：definition list <dl>、定义标题：title <dt>、定义描述：description <dd>

<dl><dt>邮箱</dt><dd>startjcu@163.com</dd><dl>

图像标签

<img src="" alt="">

图片：image <img>、资源：source src、替代文本：alternative alt

内联框架

iframe用于在一个页面中嵌入另一个页面

src：指定跳转的路径

name：必须和a标签的target属性结合使用

frameborder：用来设定边框，一般设置为0

<iframe name="my-iframe" src="#" frameborder="0"></iframe>

链接标签

锚：anchor <a>，用于页面跳转

download属性

用来指定当前链接是用来下载的，不打开文件

target属性

用来指定链接页面的打开方式

1、_blank，在新页面中打开链接

2、_self，在当前页面打开链接

3、_parent，在父级页面打开链接（有iframe的情况下才可看见效果）

4、_top，在顶级页面打开链接(有多个iframe嵌套的情况下效果可见）

href属性

用来指定跳转路径

1、相对路径，./表示当前目录，../表示上级目录

2、协议+域名+路径，//表示继承当前文件使用的协议

3、# 锚点，用于当前页面不同位置的跳转

4、？参数，用于指定参数发起请求

5、javascript伪协议，如javascript:;用于取消a标签的跳转行为

表单标签

<form>用于发起POST请求，上传数据，一个表单中必须要有一个提交按钮

提交按钮可以通过两种方式指定

1、<input type="submit" value="submit">

2、<button>submit</button>这种方式button的type属性值会默认指定为submit

输入标签

<input>用于提供输入框，选项框和提交按钮

type属性常用取值

1、text
普通输入框，输入的文本是明文

2、password
密码输入框，输入的文本是密文

3、checkbox
复选框，要设置相同的name值，且必须指定value值，供后台服务器拿数据用

4、radio
单选框，必须设置相同的name值，可以使用checked属性来设置默认选中项

通过<label>标签嵌套<input>标签的形式，可以不用指定前者的for和后者的id，精炼代码

下拉列表

<select>用于有多个可选项的情形

<option>用来指定下来列表项，selected指定默认选中项

<select><option selected>one</option></select>

Markdown语法

标题

列表
